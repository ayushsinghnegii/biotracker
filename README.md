# 🌿 BioTracker

BioTracker is an AI-powered web application that scans images of plants and animals, identifies them, and displays their scientific names along with related information.

---

# 🚀 Features

- 📷 Upload an image
- 🔍 Scan the uploaded image
- 🌿 Identify plants and animals
- 🧬 Display scientific name
- 📖 Show related information
- 📄 Generate reports
- 🛠️ Admin Dashboard
- 📱 Responsive Design
- 🌐 Share the application using ngrok

---

# 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Python
- Flask
- ngrok

---

# 📂 How to Run the Project

## Step 1

Download or clone this repository.

```bash
git clone https://github.com/ayushsinghnegii/biotracker.git
```

Move into the project folder.

```bash
cd biotracker
```

---

## Step 2

Install all required Python packages.

```bash
pip install -r requirements.txt
```

---

## Step 3

### Recommended (Windows)

Simply double-click

```
START_BIOTRACKER.bat
```

The application will start automatically.

---

### OR

Run manually

```bash
python app.py
```

---

## Step 4

Open your browser and visit

```
http://127.0.0.1:5000
```

The BioTracker application will open.

---

# 📱 Access on Mobile

If you want to use the application on your mobile phone, run

```bash
python start_share.py
```

This will generate an **ngrok** link.

Open that link on your mobile device to access BioTracker.
